# AI Code Metrics Tracking System

Full-stack system for tracking AI-augmented engineering metrics using FastMCP, VSCode extension, and FastAPI backend.

## Architecture

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   Cursor/   │────▶│ VSCode Ext    │────▶│   Backend   │
│  Windsurf   │     │ (Track Code)  │     │  FastAPI    │
└─────────────┘     └──────────────┘     └─────────────┘
                            │                      ▲
                            │                      │
                    ┌───────┴────────┐    ┌───────┴────────┐
                    │  MCP Server    │    │  JSON/JSONL     │
                    │ (Agent Logs)   │    │    Storage       │
                    └────────────────┘    └─────────────────┘
```

## Code Tracking Types

1. **completion**: Inline completion from AI (GitHub Copilot, Cursor inline)
2. **agent**: Code generated by IDE agent (MCP)
3. **manual**: Manual code written by developer

## Backend Setup

See [backend/README.md](backend/README.md) for detailed setup instructions.

Quick start:

```bash
cd backend
pip install -r requirements.txt
uvicorn src.main:app --reload
```

API will be available at http://localhost:8000

## Metrics Tracked

- Total LOC vs AI-generated LOC
- AI LOC percentage (target: 15%)
- AI-generated unit tests percentage (target: 20%)
- AI-assisted documentation percentage (target: 10%)
